# 心燈文錄 · XINDENG WENLU

私人佛典畫廊與在線讀經站。純靜態，零構建；書齋會員後端為可選的 Cloudflare Pages Functions。

- `index.html` 梵境：朝元圖橫向展卷 · 四壁畫廊 · 香煙流體 · 一百八念珠壇城環
- `canon.html` 藏經閣：漢文大藏 24 藏 4787 部全文 · 藏經櫃分函 · 豎排烏絲欄閱讀器（紙/夜/墨水屏 · 宋/楷 · 插圖）
- `admin.html` 編輯臺（隱藏頁）：文案表單 + EPUB 上架通道
- `functions/` 書齋會員後端（郵箱入館碼 · 收藏 · 書籤 · 跨設備續讀）
- `data/` 文案 · 諸藏索引 · 書目 · 全文
- 完整維護手冊見部署機桌面《無盡藏佛典网站-部署与维护手册》v2

畫作原件均為公有領域博物館級掃描；經文原版原繁體。
